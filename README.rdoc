== README

ClientChat - simple chat for Clients Support.
Worked with websocket

* Ruby version - 2.2.2

* Rails version - 4.2

* Require
    - Postgresql
    - Redis

* Database creation
    `rake db:create`

* Database initialization
    `rake db:migrate`
    `rake db:seeds` - Create Demo users

* How to run the test suite
  run `rspec` in root of rails application
